This is an incredibly simple python program that automates the SSH connection.

DO NOT REMOVE _external DIRECTORY, COULD CAUSE ISSUES. 

- executable compiled using pyinstaller --noconsole argument. If there are any errors running the program use python.exe with the bssh.py
- Supports Key Based Authentication 

1. Run the bssh.exe (made with pyinstaller) 
        - or download the source code here https://github.com/benjaminsnell/basicssh/blob/main/bssh.py.
2. Enter the IP/Domain, Username, and Check the box and select pubkey file if needed.
4. Click the "Connect" button to establish the SSH connection.
 	- OR you can also click on create, and it will create a .bat file to run for easy connection in the future. 
5. Type password if needed.
